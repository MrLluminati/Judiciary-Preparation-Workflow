# Judiciary Preparation Workflow

This repository is the structured documentation hub for Bihar Judiciary preparation.

## Purpose

This repo will maintain:

- daily study plans;
- teacher notes;
- bare-act priority notes;
- MCQ and mains-answer practice records;
- error logs;
- revision ladder tracking;
- weekly progress summaries.

## Current Operating System

The study workflow follows the two-page Bihar Judiciary wall planner:

- **Study Block 1:** Morning new learning + bare act.
- **Study Block 2:** Evening revision + MCQs/PYQ/mains answer.
- **Study Block 3:** Night light finish: Hindi, English, GK, Bihar facts, flashcards, or light law.
- **Revision ladder:** D0, D1, D7, D21, D30.
